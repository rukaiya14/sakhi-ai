## 💡 Tips for Best Recognition

1. **Speak clearly** - Enunciate each word
2. **Natural pace** - Not too fast, not too slow
3. **Quiet environment** - Reduce background noise
4. **Select correct language** - Use language dropdown
5. **Wait for "Listening"** - Don't speak too early
6. **Use keywords** - Include main words (work, job, food, etc.)

---

## 🔧 Troubleshooting

### Command not recognized?
- Make sure you selected the correct language
- Try using simpler phrases
- Use the exact keywords shown above
- Check if microphone is working

### Wrong language detected?
- Change language in dropdown
- Refresh the page
- Try again with clearer pronunciation
