## ✅ How to Find Your Sandbox Code

### Step 1: Go to Twilio Console

Open: https://console.twilio.com/us1/develop/sms/try-it-out/whatsapp-learn

### Step 2: Find Your Sandbox

You'll see a page titled **"Send a WhatsApp Message"**

Look for a section that shows:

```
┌─────────────────────────────────────────┐
│ Your Sandbox                            │
├─────────────────────────────────────────┤
│                                         │
│ To connect your phone:                  │
│                                         │
│ 1. Send this message to +1 415 523 8886│
│                                         │
│    join <your-code-here>                │
│                                         │
│ Example: join happy-tiger               │
│          join mile-rhyme                │
│          join clever-moon               │
└─────────────────────────────────────────┘
```

### Step 3: Copy Your Code

Your code will be something like:
- `happy-tiger`
- `mile-rhyme`
- `clever-moon`
- `brave-lion`
- etc.

It's **TWO WORDS** separated by a hyphen.

### Step 4: Send Join Message

1. Open WhatsApp
2. Send to: `+1 415 523 8886`
3. Message: `join <your-code>`

Example:
```
join happy-tiger
```

### Step 5: Wait for Confirmation

You should receive:
```
✅ Awesome, you are all set!

Your Sandbox: join happy-tiger
Reply stop to leave this Sandbox
```

## 🔍 What to Look For

Your sandbox code is:
- ✅ Two words (e.g., `happy-tiger`)
- ✅ Separated by hyphen (-)
- ✅ All lowercase
- ✅ Shown in Twilio Console

NOT:
- ❌ Long alphanumeric string
- ❌ Account SID (starts with AC)
- ❌ Auth Token
- ❌ Phone number
