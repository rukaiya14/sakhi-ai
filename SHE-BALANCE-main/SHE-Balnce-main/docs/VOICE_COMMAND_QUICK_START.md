# Voice Command Quick Start Guide

## ✅ Working Now - No Setup Required!

The voice command system uses your browser's built-in speech recognition. It works immediately without any AWS setup or backend configuration.

## How to Test

### Option 1: Test Page
1. Open the test page:
   ```
   SHE-BALANCE-main/SHE-Balnce-main/test-voice-command.html
   ```

2. Select your language from the dropdown

3. Click the microphone button

4. Speak a command (e.g., "Dashboard", "डैशबोर्ड", "AI Sakhi")

5. The system will:
   - Detect your language automatically
   - Show what it heard
   - Speak confirmation in your language
   - Navigate to the page

### Option 2: Dashboard
1. Open the dashboard:
   ```
   SHE-BALANCE-main/SHE-Balnce-main/dashboard.html
   ```

2. Click the microphone button in the header

3. Speak your command

## How It Works

1. **Click Microphone** → Button turns red, starts listening
2. **Speak Command** → Browser captures your voice
3. **Recognition** → Browser converts speech to text in your language
4. **Matching** → System matches command to navigation
5. **Confirmation** → Speaks confirmation in your language
6. **Navigation** → Automatically opens the page

## Troubleshooting

### "Microphone access denied"
**Solution:** Enable microphone permissions in browser settings
- Chrome: Settings → Privacy → Site Settings → Microphone
- Edge: Settings → Cookies and site permissions → Microphone

### "No speech detected"
**Solution:** 
- Speak louder and clearer
- Check microphone is working
- Reduce background noise
- Try again

### "Command not recognized"
**Solution:**
- Use exact commands from the list above
- Speak clearly and slowly
- Check language is set correctly
- Try simpler commands like "Dashboard" or "Skills"

### Button not responding
**Solution:**
- Refresh the page
- Check browser console for errors (F12)
- Try a different browser (Chrome recommended)

## Tips for Best Results

1. **Speak Clearly** - Enunciate words properly
2. **Reduce Noise** - Find a quiet environment
3. **Use Simple Commands** - Single words work best
4. **Wait for Button** - Let it turn red before speaking
5. **Check Language** - Make sure correct language is selected
